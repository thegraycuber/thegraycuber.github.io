  index/ - my main page, which allows for navigation between all projects in the current framework
  media/ - media
  
  each other folder is a project in the current framework - some visualization of a math topic.

  the html files in the highest level are redirecting to the folders. i did this so that people would have some warning time before i break any links. these will be deleted soon.
